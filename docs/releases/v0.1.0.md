# v0.1.0

## 摘要
- 這是 `d2r-hyper-launcher` 的第一個正式版標籤發行。
- 建立 Windows D2R 玩家可直接使用的多開啟動流程、首次使用引導，以及視窗切換功能。
- 整合近期多開體驗修正，讓玩家在啟動多帳號時有更清楚的提示與更安全的流程。

## 重點
- 第一次執行會自動建立 `accounts.csv`，離開時會自動開啟資料夾，並確保 CSV 可正常被 Excel 開啟。
- 玩家可以直接在工具裡選擇正確的 `D2R.exe` 路徑，不需要手動改設定檔。
- 批次啟動流程改成先檢查目前帳號狀態，再提供更清楚的已啟動 / 未啟動資訊與更合理的等待節奏。
- README 與使用文件已整理成以玩家操作流程為主的說明。

## 各範圍變更

### multiboxing
- 新增更安全的首次帳號設定流程：自動建立 `accounts.csv`、提示玩家修改內容，並處理 UTF-8 BOM 相容性。
- 將批次啟動預設等待時間調整為 30 秒，降低短時間重複連線帶來的風險。
- 單帳號、批次與離線模式都會先確認 `D2R.exe` 路徑有效，再進入啟動流程。
- 單帳號啟動已補齊 mod 選擇流程，並且在該帳號已開啟時阻止重複啟動。
- 批次啟動會先預掃描帳號清單，明確列出 `[已啟動]` / `[未啟動]`，若全部已啟動就不再多問後續選項。
- 補充提醒：主選單狀態判斷依賴 `DisplayName` 對應目前視窗標題。

### switcher
- 納入內建視窗切換流程，並補充說明：只有在 launcher 持續開著時，切窗功能才會生效。

### 文件 / workflow
- 重寫玩家快速開始說明，改成下載、首次執行、修改帳號檔、回到工具使用的順序。
- 補充登入卡住、批次啟動節奏、DisplayName 狀態偵測等常見注意事項。
- 加入 Windows 環境下較安全的驗證與發版流程說明。

## 升級注意事項
- 這是第一個正式 release，因此沒有前一版 tag 可供比較。
- 若你依賴主選單的已啟動 / 未啟動狀態，請確認 `accounts.csv` 裡的 `DisplayName` 與目前遊戲視窗標題一致。
- 如果你的 D2R 不在預設安裝位置，請在工具內使用 `p` 選擇正確的 `D2R.exe`。
